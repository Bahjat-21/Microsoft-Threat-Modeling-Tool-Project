# Industrial Threat Modeling Project

This project contains a cybersecurity threat model for an industrial OT/IT environment created with a Threat Modeling Tool (`.tm7`).

## Overview

The model represents a production network including:

- ERP / MES systems
- OPC Server
- PLCs
- HMI stations
- CNC machines
- Firewalls and VPN access
- Engineering stations
- NAS and file servers

The communication architecture includes industrial and IT protocols such as:

- OPC UA
- Modbus TCP
- PROFINET
- SMB
- HTTP
- RDP
- IPSec

## Purpose

The goal of this project is to analyze cybersecurity risks in an industrial environment according to IEC 62443 principles.

The threat model identifies:

- attack vectors
- vulnerabilities
- network risks
- environmental threats
- security controls and mitigations

## Included Security Topics

- Malware protection
- Authentication and authorization
- Data integrity
- Encryption
- Replay protection
- DoS protection
- Physical security
- Environmental hazards

## Standards

This project references security requirements from:

- IEC 62443
- Industrial Control System (ICS) security concepts

## File Type

`.tm7` files are Threat Modeling Tool project files containing:

- architecture diagrams
- trust boundaries
- data flows
- threat instances
- mitigation controls

## Educational Purpose

This repository was created for educational and cybersecurity analysis purposes.
